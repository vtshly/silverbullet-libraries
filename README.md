# SilverBullet Libraries

This is my library of `space-lua` commands, functions and snippets for [SilverBullet](https://silverbullet.md)

- SilverBullet is a note-taking tool capable of running Lua code right inside markdown files

## Install

1. Run command in SB `Library: Add Repository` 
2. --> insert repo url `https://github.com/vtshly/silverbullet-libraries/blob/main/Repository/vtshly.md`
3. Install the desired libraries via the `Library: Manager`
